## Getting Started

Clone repository

```bash
# Using SSH
git clone git@github.com:laksby/quickmerge.git

# Using HTTPS
git clone https://github.com/laksby/quickmerge.git
```

Setup project

```bash
npm install
```

Start project locally

```bash
npm start
```
